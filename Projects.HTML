Virtual Memory Project
In this project I created a virtual memory simulation in Java. My goal was to find the algorithm that would be the best at avoiding page faults. The page table we simulated was a 32 bit address space with a page size of 2KB. After conducting simulations on these four different algorithms: Rand, Clock, NRU, Opt, there is one algorithm that is clearly the best at avoiding page faults. Across all frame numbers Opt performed the best. The algorithms we used we as follows: Opt Simulate what the optimal page replacement algorithm would choose if it had perfect knowledge. Rand Choose a valid page at random. Clock Use the better implementation of the second-chance algorithm NRU Pick a not recently used page

Auto Complete Project
In this project I implemented a De La Braindias Trie using Java that stores words implicitly through keys. I u sed a recursive algorithm to identify the correct placement of a letter to create a defacto dictionary. As the user in these systems type in letters, the system shows an updated list of word suggestions to help the user complete the word thy are typing. The system will order the suggested words based on a _score_ that represents the number of times a word has been selected as a suggestion. The system can learn new words that are not in its dictionary.
